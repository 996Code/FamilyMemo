# FamilyMemo - 产品需求文档（PRD）

> 版本 v0.2 | 2026-03-20

---

## 一、产品定位

面向家庭成员的协作备忘录工具。解决家庭日常事务提醒分散、信息传达不及时的问题。
核心价值：一个家庭群组，所有人都能看到、被提醒、互相@。

---

## 二、目标用户

- 主要：有子女的家庭（父母 + 子女，2~6 人）
- 场景：买菜提醒、接送孩子、缴费、家庭活动等日常事务

## 四、登录方案

### 鸿蒙端
- 使用华为账号 OAuth（Account Kit），一键授权，获取 openId 作为用户唯一标识
- 正式阶段后端用 openId 关联用户记录

### iOS 端
- 使用 Sign in with Apple（苹果强制要求），获取 userIdentifier 作为唯一标识
- 正式阶段后端用 userIdentifier 关联用户记录

### 统一后端处理
- 后端收到各平台 token 后验证合法性，返回统一 JWT
- 用户表记录：id / platform(huawei|apple) / platformUserId / name / avatar / deviceToken

## 五、群组加入方式（二维码）

### 流程
1. 群主在群组详情页点击「邀请成员」
2. 后端生成含 inviteToken 的链接，前端渲染为二维码
3. 受邀者用应用内「扫码加入」扫描二维码
4. 后端验证 inviteToken（有效期 24 小时），将用户加入群组
5. 加入成功后跳转到群组页面

### MVP 简化版
- MVP 阶段不做真实二维码，用文字邀请码（6位字母数字）替代
- 正式阶段升级为二维码

## 七、UI 风格定义

### 整体风格
- 简洁卡片式，圆角大（16~20dp），留白充足
- 主色调：柔和蓝（#4A90E2），辅助暖橙（#E27B4A）
- 背景：浅灰白（#F5F7FA，与启动页一致）
- 字体：系统默认，标题 18sp Bold，正文 14sp，辅助 12sp Gray

### 圆形设计元素
- 用户头像：圆形（带主题色边框）
- 通知角标：圆形红点
- FAB 按钮：圆形
- 成员头像组（群组卡片）：小圆形叠加排列
- MemberChip（@提及标签）：圆角胶囊形
- 完成状态：圆形复选框

### 配色规则
- 每位用户有专属主题色（User.color 字段）
- 用户头像圆圈边框使用该主题色
- @提及的 MemberChip 背景使用该用户主题色（低透明度）
