# FamilyMemo

> 面向家庭成员的协作备忘录 App，支持多人群组、@提及、定时提醒推送。

---

## 项目状态

**当前阶段：Phase 1 MVP — 鸿蒙端（内存数据库）**

| 平台 | 状态 |
|---|---|
| HarmonyOS | 开发中 |
| iOS | 规划中（Phase 4）|
| 后端服务 | 规划中（Phase 2）|

---

## 核心功能

- **群组协作**：家庭成员组建群组，共享备忘录
- **@提及**：创建备忘录时 @成员，被提及者收到通知
- **定时提醒**：设置提醒时间，到时手机推送通知
- **二维码邀请**：扫码加入群组（Phase 3）
- **多端支持**：鸿蒙 + iOS，数据层可无缝切换
